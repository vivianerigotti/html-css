Principais formatações

Negrito / Destaque

Nesta frase, temos um termo em negrito usando a tag B (sem semântica).

Nesta frase, temos um termo em destaque usando a tag STRONG (com semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag I (sem semântica)

Nesta frase, temos um termo em ênfase usando a tag EM (com semântica)

Texto marcado

Podemos criar também um texto marcado usando a tag MARK.

E neste parágrafo temos outro texto marcado no final.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado, com a tag DEL.

Texto inserido e deletado

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, com a tag INS (com semântica).

Existe também o sublinhado om a tag U (sem semântica).

Texto sobrescrito

Para inserir coisas do tipo x20 + 3, usamos a tag SUP.

Texto subscrito

Para inserir coisas do tipo H2O, usamos a tag SUB.
